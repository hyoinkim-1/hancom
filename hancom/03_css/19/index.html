순서 있는 리스트 <ol>HTML

<ol> 태그는 순서대로 나열되는 리스트입니다. type 속성으로 "1"(숫자), "i"(로마숫자), "a"(알파벳) 등으로 변경 가능하고, start 속성으로 시작 번호를 설정할 수 있습니다.

테이블 구조 <table>HTML

<thead>는 표 제목, <tbody>는 메인 데이터를 담습니다. colspan(열 병합), rowspan(행 병합)으로 셀을 확장할 수 있고, scope 속성으로 접근성을 높일 수 있습니다.

폼 태그 <form>HTML

action(데이터 처리 URI), method(post/get), target(_self/_blank) 속성으로 폼 동작을 제어합니다. <input>, <label>, <button> 등의 요소를 포함하여 사용자 입력을 처리합니다.

iframe 임베딩HTML

<iframe>은 다른 HTML 페이지를 현재 페이지에 임베딩합니다. 유튜브 영상이나 지도 같은 외부 콘텐츠를 삽입할 때 주로 사용됩니다.

Class vs IDHTML

class는 여러 요소를 그룹핑할 수 있고(.selector), id는 유일한 요소에만 사용합니다(#selector). JavaScript의 getElementById, querySelector로 요소를 선택할 수 있습니다.

br, hr, details 태그HTML

<br>은 줄바꿈, <hr>은 가로줄을 표시합니다. <details>는 접었다 펼칠 수 있는 드롭박스 요소이며, open 속성으로 기본 상태를 제어할 수 있습니다.

비디오/오디오 태그HTML

<video>, <audio> 태그로 미디어를 임베딩합니다. src 속성 대신 <source>를 사용하면 여러 형식을 지원할 수 있습니다. controls, autoplay 속성으로 재생 컨트롤을 제어합니다.

VSCode EmmetHTML

Emmet으로 HTML을 빠르게 생성합니다. 자식(>), 반복(*N), 클래스(.), ID(#), 속성([]), 텍스트({}) 등의 기호를 조합하여 복잡한 구조를 한 줄로 작성할 수 있습니다.

웹 접근성 속성Accessibility

aria-*, role, alt, lang 속성으로 웹 접근성을 높입니다. lang을 특정 문단에 지정하면 스크린 리더가 올바른 언어로 읽을 수 있습니다. 숨김 처리도 접근성 측면에서 중요합니다.

CSS 기본 문법CSS

선택자 { 속성: 값; } 형식입니다. 작성 순서는 body(전체) → 요소 → 클래스 → 자손 순입니다. 중복되는 경우 마지막에 작성된 CSS가 적용됩니다.

Claude(클코) 기본 사용법Tool

Shift+Tab으로 모드 전환, @fileName으로 파일 언급, /clear로 세션 정리, /rename으로 세션명 변경 등의 명령어를 사용합니다. /effort로 작업 효율을 지정할 수 있습니다.

StylelintCSS Tool

CSS 코드 품질을 검사하는 린터입니다. npm install로 설치하고 npx stylelint로 실행합니다. --fix 옵션으로 자동 수정이 가능하여 코드 일관성을 유지할 수 있습니다.

박스 모델CSS

Padding(내부 여백), Margin(외부 여백), Border(테두리)로 이루어집니다. 각각 위치별로 설정할 수 있고, border-style은 solid, dashed 등 다양한 스타일을 지원합니다.

Display: FlexCSS

display: flex로 자식 요소의 배치를 자유롭게 제어합니다. gap으로 자식 요소 간 간격을 설정하고, justify-content, align-items로 정렬을 제어할 수 있습니다.

CSS 적용 방법CSS

외부 파일(link), 내부(<style>), 인라인(style 속성) 3가지 방식이 있습니다. 우선순위는 인라인 > 내부/외부입니다. 외부 파일 방식이 유지보수에 가장 좋습니다.

가상클래스 (LVHA)CSS

:link(미방문) → :visited(방문) → :hover(마우스 올림) → :active(클릭 중) 순서대로 작성해야 합니다. 나중에 작성된 CSS가 먼저 작성된 것을 덮으므로 순서가 중요합니다.

Position 속성CSS

static(기본), relative(자신 기준), absolute(부모 기준), fixed(뷰포트 기준), sticky(임계값 넘으면 고정)입니다. 겹쳐 띄울 때는 부모 relative + 자식 absolute를 사용합니다.

Grid 레이아웃CSS

display: grid로 격자 레이아웃을 만듭니다. grid-template-columns: repeat(2, 1fr 2fr)으로 열을 나눕니다. fr을 사용하는 이유는 gap을 제외한 영역에서만 분할하기 때문입니다.

Transition 및 TransformCSS

transition으로 요소 변화의 속도를 부드럽게 조절합니다. transform-style: preserve-3d로 입체공간을 표현하고, background-image로 선형/방사형 그라데이션을 적용합니다.

@media 쿼리CSS

@media (min-width: 30em) and (max-width: 50em)로 반응형 디자인을 구현합니다. 기기의 width/height에 따라 다른 CSS를 적용하여 모든 화면 크기에서 최적의 레이아웃을 제공합니다.

flex-directionCSS

flex-direction으로 flex 요소들이 쌓이는 방향을 조절합니다. row(가로), column(세로), row-reverse, column-reverse 등의 옵션이 있어 레이아웃 유연성을 높입니다.

CSS 변수CSS

:root에 --main-bgc 같은 변수를 정의하고 var(--main-bgc)로 사용합니다. 컬러팔레트를 전역변수로 설정하면 개별 색상 수정이 아닌 변수만 수정하여 유지보수성을 높입니다.

z-index 작동 원리CSS

z-index는 static이 아닌 position(relative, absolute 등)에만 적용됩니다. 따라서 position: relative를 먼저 지정해야 z-index가 작동합니다.

가상클래스 vs 가상요소CSS

가상클래스(:hover)와 가상요소(::after)는 다릅니다. 작성 순서는 요소:가상클래스::가상요소입니다. ::after는 실제 DOM이 아니므로 :hover 같은 이벤트를 발생시킬 수 없습니다.

CSS ResetCSS

브라우저의 기본 padding, margin, box-sizing 설정을 초기화합니다. *{ padding: 0; margin: 0; box-sizing: border-box; }로 모든 요소를 리셋하여 일관된 디자인을 구현합니다.
